# GitHub Basic Practice

GitHub Basic ワークショップで使う、個人用の練習リポジトリです。

このリポジトリには、ブロックが盤面の底で止まらない **Falling Blocks** アプリが入っています。最初からバグを直さず、次の順番で作業してください。

1. Review相手をCollaboratorへ招待する
2. IssueにMarkdownで目的と完了条件を書き、Label / Assignee / @メンションを設定する
3. `main` から作業 Branch を作る
4. `app/falling-blocks/game.js` の `FLOOR_ROW` を修正する
5. Commit / PushしてPull Requestを作る
6. ペアでReviewしてからMergeする
7. Merge後にIssue、履歴、`git blame`を確認する

## 修正対象

```text
app/falling-blocks/game.js
```

正解を先に見ず、まず `app/falling-blocks/index.html` をブラウザで開いて不具合を確認してください。
