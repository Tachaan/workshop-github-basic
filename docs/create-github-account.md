# GitHub アカウントを作成する

> ℹ️ 本ワークショップの前提事項は「GitHub アカウントを持っていること」です。
> まだ持っていない場合は、このページの手順で個人用アカウントを作成してください。

## 1. 作成手順（概要）

1. ブラウザで <https://github.com/> を開く
2. **Sign up（サインアップ）** を選ぶ
3. 画面の案内に従ってメールアドレス・パスワード・ユーザー名を入力する
4. 届いた確認メールでメールアドレスを検証する

> 🔑 **ポイント**: メールアドレスの検証を完了しないと、リポジトリ作成などの基本操作ができません。必ず確認メールのリンクを開いてください。

## 3. 作成後におすすめの設定

- **2要素認証(2FA)** を設定する（アカウントのセキュリティが大きく向上します）
- プロフィールの表示名を設定しておく（ハンズオンでレビューする際に誰かわかりやすくなります）

2FA は強く推奨されますが、当日の進行で設定に時間がかかる場合は講師の案内に従ってください。
アカウント作成やメール検証で詰まった場合も、その場で講師に相談してください。
